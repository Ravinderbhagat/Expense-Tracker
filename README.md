# 💸 Expense Tracker

A responsive browser-based Expense Tracker built with HTML, CSS, and JavaScript.

🔗 **Live Demo:** https://ravinderbhagat.github.io/Expense-Tracker/

---

## Features

- Add and delete expenses with amount, date, category, and description
- Monthly budget tracker with a live progress bar
- Summary cards for total spent, current month, transaction count, and average expense
- Category-wise doughnut chart using Chart.js
- Browse spending history by month
- Filter expenses by category
- Sort expenses by newest, oldest, highest or lowest amount
- Search expenses by keyword
- Quick-add buttons for common daily expenses
- Export expense data as a CSV file
- Data saved locally using localStorage
- Responsive layout for desktop and mobile

---

## Tech Stack

- HTML5
- CSS3
- JavaScript ES6
- Chart.js
- localStorage
- Google Fonts

---

## Getting Started

No installation needed. Just open the live link above in any browser.

To run locally:
1. Download or clone this repository
2. Open `index.html` in your browser
